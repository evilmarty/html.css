Usage

To use html.css is very easy, just write regular HTML. You can also use CSS classes for specialised or custom components.

<p>This is <strong>text</strong> and this is <small>text</small></p>

Can also be written like this:

<div class="p">This is <span class="strong">text</span> and this is <span class="small">text</span></div>

<div> and <span> tags don't have any styles applied to them, so they can be customised however you like.
